# .NET for code analysis

The following summarizes .NET support for code analysis.

**Frameworks**:

* .NET Framework 4.6-4.8.x
* ASP.NET 6.x (C# only)
* .NET 6

**Import your app through SCM**: Available

**Test or monitor your app through CLI and IDE**: Available

**Features**:

* Reports
* Custom rules
* Interfile analysis
